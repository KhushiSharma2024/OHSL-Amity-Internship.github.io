About Us

The OHSL Summer Internship 2024, in partnership with Amity University Noida, empowered students to innovate in AI-driven solutions. Guided by mentor Khushi Sharma, participants harnessed technologies like Retrieval-Augmented Generation (RAG), Large Language Models (LLMs), and the Falcon-7B model to advance search engines, document processing, and email automation.

Interactive Search Engine Using RAG and LLMs

This project created a dynamic search engine powered by Retrieval-Augmented Generation (RAG) and Large Language Models (LLMs). It enables real-time document retrieval and generates precise, context-aware responses, optimizing user experience and search performance.

Features

  • Real-time document retrieval
  • Contextual response generation using LLMs
  • Intuitive and user-friendly interface

Document Processing Question Answering System Using Falcon-7B and RAG

This system processes documents and delivers accurate answers using the Falcon-7B model and RAG. It efficiently parses and indexes documents, providing context-aware responses to user queries.

Features

  • Efficient document parsing and indexing
  • Advanced question-answering with Falcon-7B
